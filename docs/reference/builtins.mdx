# Built-in Functions

Setsuna provides a set of built-in functions for common operations. These are available globally without any imports.

## Overview

| Category | Functions |
|----------|-----------|
| **I/O** | `print`, `println`, `input`, `input_prompt` |
| **File I/O** | `file_read`, `file_write`, `file_append`, `file_exists`, `file_delete`, `file_lines`, `dir_list`, `dir_exists` |
| **Type Conversion** | `str`, `int`, `float` |
| **Type Checking** | `is_int`, `is_float`, `is_string`, `is_bool`, `is_list`, `is_tuple`, `is_record`, `is_fn` |
| **List Operations** | `head`, `tail`, `cons`, `len`, `empty`, `reverse`, `append`, `concat`, `nth`, `range`, `sort` |
| **Math** | `abs`, `sqrt`, `pow`, `min`, `max`, `floor`, `ceil`, `round` |
| **Extended Math** | `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `atan2`, `log`, `log10`, `exp`, `random`, `random_int`, `pi`, `e` |
| **Strings** | `substr`, `split`, `join`, `uppercase`, `lowercase`, `trim`, `trim_start`, `trim_end`, `contains`, `starts_with`, `ends_with`, `replace`, `replace_all`, `char_at`, `chars`, `index_of` |
| **Utility** | `error`, `assert`, `compare` |
| **Standard Library** | `map`, `filter`, `fold`, `reduce`, `take`, `drop`, `zip`, `any`, `all`, `find`, `flatten`, `sum`, `product`, `maximum`, `minimum`, `compose`, `pipe`, and more... |

<Note>
The Standard Library functions are loaded automatically from `stdlib/prelude.stsn`. See [List Operations](/reference/list-ops) for details.
</Note>

## I/O Functions

### print

Output a value to the console.

```setsuna
print("Hello, World!")     // Hello, World!
print(42)                  // 42
print([1, 2, 3])           // [1, 2, 3]
print({ name: "Alice" })   // { name: "Alice" }
```

**Signature:** `print(value) -> ()`

### println

Alias for `print`. Output a value to the console with a newline.

```setsuna
println("Hello, World!")
```

**Signature:** `println(value) -> ()`

### input

Read a line from standard input (no prompt).

```setsuna
let name = input()
print("Hello, " + name)
```

**Signature:** `input() -> String`

### input_prompt

Display a prompt and read a line from standard input.

```setsuna
let name = input_prompt("Enter your name: ")
print("Hello, " + name)
```

**Signature:** `input_prompt(prompt) -> String`

## Type Conversion

### str

Convert any value to its string representation.

```setsuna
str(42)        // "42"
str(3.14)      // "3.14"
str(true)      // "true"
str([1, 2])    // "[1, 2]"
```

**Signature:** `str(value) -> String`

### int

Convert a value to an integer.

```setsuna
int("42")      // 42
int(3.7)       // 3
int(true)      // 1
int(false)     // 0
```

**Signature:** `int(value) -> Int`

### float

Convert a value to a floating-point number.

```setsuna
float("3.14")  // 3.14
float(42)      // 42.0
```

**Signature:** `float(value) -> Float`

## Type Checking

Functions to check the type of a value at runtime.

### is_int

```setsuna
is_int(42)       // true
is_int(3.14)     // false
is_int("42")     // false
```

**Signature:** `is_int(value) -> Bool`

### is_float

```setsuna
is_float(3.14)   // true
is_float(42)     // false
```

**Signature:** `is_float(value) -> Bool`

### is_string

```setsuna
is_string("hello")  // true
is_string(42)       // false
```

**Signature:** `is_string(value) -> Bool`

### is_bool

```setsuna
is_bool(true)    // true
is_bool(false)   // true
is_bool(1)       // false
```

**Signature:** `is_bool(value) -> Bool`

### is_list

```setsuna
is_list([1, 2, 3])  // true
is_list([])         // true
is_list((1, 2))     // false
```

**Signature:** `is_list(value) -> Bool`

### is_tuple

```setsuna
is_tuple((1, 2))    // true
is_tuple([1, 2])    // false
```

**Signature:** `is_tuple(value) -> Bool`

### is_record

```setsuna
is_record({ x: 1 })  // true
is_record([1, 2])    // false
```

**Signature:** `is_record(value) -> Bool`

### is_fn

Check if a value is a function (closure or builtin).

```setsuna
is_fn((x) => x * 2)  // true
is_fn(print)         // true
is_fn(42)            // false
```

**Signature:** `is_fn(value) -> Bool`

## List Operations

### head

Get the first element of a list.

```setsuna
head([1, 2, 3])    // 1
head(["a", "b"])   // "a"
head([])           // Error: empty list
```

**Signature:** `head(list) -> T`

### tail

Get all elements except the first.

```setsuna
tail([1, 2, 3])    // [2, 3]
tail([1])          // []
tail([])           // Error: empty list
```

**Signature:** `tail(list) -> List<T>`

### cons

Prepend an element to a list.

```setsuna
cons(1, [2, 3])        // [1, 2, 3]
cons("a", [])          // ["a"]
cons(0, cons(1, []))   // [0, 1]
```

**Signature:** `cons(element, list) -> List<T>`

### len

Get the length of a list or string.

```setsuna
len([1, 2, 3])     // 3
len([])            // 0
len("hello")       // 5
```

**Signature:** `len(collection) -> Int`

### empty

Check if a list is empty.

```setsuna
empty([])          // true
empty([1, 2])      // false
```

**Signature:** `empty(list) -> Bool`

### reverse

Reverse a list.

```setsuna
reverse([1, 2, 3])     // [3, 2, 1]
reverse([])            // []
reverse(["a", "b"])    // ["b", "a"]
```

**Signature:** `reverse(list) -> List<T>`

### append

Append an element to the end of a list.

```setsuna
append([1, 2], 3)          // [1, 2, 3]
append([], 1)              // [1]
```

**Signature:** `append(list, element) -> List<T>`

### concat

Concatenate two lists.

```setsuna
concat([1, 2], [3, 4])     // [1, 2, 3, 4]
concat([], [1, 2])         // [1, 2]
concat([1], [])            // [1]
```

**Signature:** `concat(list1, list2) -> List<T>`

### nth

Get the element at a specific index (0-based).

```setsuna
nth([10, 20, 30], 0)       // 10
nth([10, 20, 30], 2)       // 30
nth([10, 20, 30], 5)       // Error: index out of bounds
```

**Signature:** `nth(list, index) -> T`

### range

Generate a list of integers from start (inclusive) to end (exclusive).

```setsuna
range(0, 5)                // [0, 1, 2, 3, 4]
range(1, 4)                // [1, 2, 3]
range(5, 5)                // []
```

**Signature:** `range(start, end) -> List<Int>`

### sort

Sort a list of numbers or strings in ascending order.

```setsuna
sort([3, 1, 4, 1, 5])      // [1, 1, 3, 4, 5]
sort(["c", "a", "b"])      // ["a", "b", "c"]
sort([])                   // []
```

**Signature:** `sort(list) -> List<T>`

## Math Functions

### abs

Absolute value.

```setsuna
abs(-5)      // 5
abs(5)       // 5
abs(-3.14)   // 3.14
```

**Signature:** `abs(number) -> Number`

### sqrt

Square root.

```setsuna
sqrt(16)     // 4.0
sqrt(2)      // 1.41421...
```

**Signature:** `sqrt(number) -> Float`

### pow

Exponentiation.

```setsuna
pow(2, 3)    // 8
pow(2, 0.5)  // 1.41421...
pow(10, -1)  // 0.1
```

**Signature:** `pow(base, exponent) -> Number`

### min

Minimum of two values.

```setsuna
min(3, 5)      // 3
min(-1, 1)     // -1
min(3.14, 2.71)  // 2.71
```

**Signature:** `min(a, b) -> Number`

### max

Maximum of two values.

```setsuna
max(3, 5)      // 5
max(-1, 1)     // 1
max(3.14, 2.71)  // 3.14
```

**Signature:** `max(a, b) -> Number`

### floor

Round down to nearest integer.

```setsuna
floor(3.7)     // 3
floor(-3.2)    // -4
floor(5.0)     // 5
```

**Signature:** `floor(number) -> Int`

### ceil

Round up to nearest integer.

```setsuna
ceil(3.2)      // 4
ceil(-3.7)     // -3
ceil(5.0)      // 5
```

**Signature:** `ceil(number) -> Int`

### round

Round to nearest integer.

```setsuna
round(3.4)     // 3
round(3.5)     // 4
round(-3.5)    // -4
```

**Signature:** `round(number) -> Int`

## Utility Functions

### type

Get the type of a value as a string.

```setsuna
type(42)           // "Int"
type(3.14)         // "Float"
type("hello")      // "String"
type(true)         // "Bool"
type([1, 2, 3])    // "List"
type((1, 2))       // "Tuple"
type({ x: 1 })     // "Record"
type((x) => x)     // "Function"
```

**Signature:** `type(value) -> String`

### error

Throw a runtime error with a message.

```setsuna
fn divide(a, b) {
    if b == 0 {
        error("Division by zero")
    } else {
        a / b
    }
}
```

**Signature:** `error(message) -> Never`

### assert

Assert that a condition is true, or throw an error with a message.

```setsuna
assert(x > 0, "x must be positive")
assert(len(list) > 0, "list cannot be empty")
```

**Signature:** `assert(condition, message) -> ()`

### compare

Compare two values, returning -1, 0, or 1.

```setsuna
compare(1, 2)        // -1
compare(2, 2)        // 0
compare(3, 2)        // 1
compare("a", "b")    // -1
compare("b", "a")    // 1
```

**Signature:** `compare(a, b) -> Int`
